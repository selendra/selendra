### Burn Mechanism (Deflationary Pressure)

**Two Types of Burns**:

1. **Initial Burn (Year 0)**: **90M SEL burned immediately**

   - Non-recoverable, provable on-chain

2. **Ongoing Fee Burn**: **30% of all transaction fees burned continuously**
   - Creates perpetual deflationary pressure
   - Offsets inflation over time
   - Same mechanism as Ethereum EIP-1559

**Projected Fee Burn (Balanced Model)**:
- Year 1: $88K revenue × 30% burn = $26.4K burned ≈ 169K SEL @ $0.156

**Cumulative Fee Burn Projection**:
- Year 1-10: ~2.5M SEL burned (fees)
- Year 11-50: ~30M SEL burned (fees)
- Year 51-100: ~47.5M SEL burned (fees)
- **Total Fee Burn (100 years)**: ~80M SEL

**Net Supply at Year 100**:
- Initial: 140M (after initial burn)
- Minted: +260M (Year 1-100 emissions)
- Fee burns: -80M (ongoing burns)
- **Net Supply**: ~320M SEL (vs 400M theoretical max)

**Total Burn Impact (Year 100)**:
- Initial burn: 90M SEL
- Fee burns: 80M SEL
- **Total burned**: ~170M SEL
- **Original supply**: 229.67M
- **Final supply**: ~320M (starting from 400M max)

This creates **sustainable deflationary** pressure, with total burns (170M) balancing long-term emissions while maintaining growth capacity.

## Value Accrual Mechanisms

### 1. Transaction Fee Model

**Fee Structure** (follows Ethereum EIP-1559):
- **Base Fee**: Dynamic, algorithmically adjusted based on network congestion
  - Target: 50% block fullness
  - Adjustment: ±12.5% per block
  - Average: $0.001-$0.005 per transaction
- **Priority Fee (Tip)**: User-set, goes to validators (MEV capture)
  - Average: 10-20% of base fee

**Fee Distribution**:
```
Total Transaction Fee = Base Fee + Priority Fee

Base Fee:
  30% → Burned (deflationary pressure)
  20% → Treasury (protocol development)
  50% → Validators & Stakers (security incentive)

Priority Fee:
  100% → Block Producer (MEV reward, tip)
```

**Example Transaction**:
- Base fee: $0.003
- Priority fee: $0.0003
- Total: $0.0033

Distribution:
- Burned: $0.0009 (30% of base)
- Treasury: $0.0006 (20% of base)
- Stakers: $0.0015 (50% of base)
- Block producer: $0.0003 (100% of priority)

```rust
// Target staking rate: 70%
// If actual staking rate < 70%, increase rewards
// If actual staking rate > 70%, decrease rewards

fn calculate_inflation(current_staking_rate: Percent) -> Percent {
    const TARGET_STAKING: Percent = Percent::from_percent(70);
    const MIN_INFLATION: Percent = Percent::from_percent(2);
    const MAX_INFLATION: Percent = Percent::from_percent(10);

    if current_staking_rate < TARGET_STAKING {
        // Increase inflation to incentivize staking
        min(MAX_INFLATION, current_inflation * 1.1)
    } else {
        // Decrease inflation if over-staked
        max(MIN_INFLATION, current_inflation * 0.95)
    }
}
```

- Reduces reliance on fixed inflation schedule
- Adapts to market conditions
