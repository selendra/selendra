# Getting Started with Selendra

Welcome to Selendra! This guide will help you get started with the Selendra blockchain.

## What is Selendra?

Selendra is a high-performance blockchain built with Substrate, featuring 1-second block times and sub-2-second finality.

## Quick Start

### Prerequisites

- Node.js 18+
- A web3 wallet (MetaMask, Polkadot.js extension)

### Connect to Mainnet

Selendra mainnet is now live! Connect to the network using:

- **RPC Endpoint**: `wss://rpc.selendra.org`
