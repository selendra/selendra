# Network Overview

## Network Specifications

- **Block Time**: 1 second (Aura consensus)
- **Finality**: <2 seconds (AlephBFT)
- **Consensus**: Hybrid Aura + AlephBFT
- **Framework**: Substrate
- **Status**: Mainnet Live

## Key Features

### Fast Finality
Selendra achieves sub-2-second finality using AlephBFT, ensuring transactions are irreversible quickly.

### EVM Compatibility
Full Ethereum compatibility through Frontier, allowing seamless deployment of existing dApps.

### Cross-Chain Ready
Built on Substrate, Selendra is prepared for future Polkadot parachain integration.
